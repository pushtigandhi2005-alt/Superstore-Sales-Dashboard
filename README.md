# 📊 Superstore Sales Dashboard

## Project Overview
This project is an interactive Power BI Sales Dashboard created using the Sample Superstore dataset. It provides insights into sales performance, profit, orders, and profit margin through interactive visualizations and KPI cards.

## Features
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Profit Margin Measure (DAX)
- Interactive Dashboard
- Sales Performance Analysis

## Tools Used
- Microsoft Power BI
- DAX (Data Analysis Expressions)
- Sample Superstore Dataset

## Dashboard Insights
- Analyzed overall sales and profit performance.
- Calculated Profit Margin using a DAX measure.
- Visualized key business metrics using KPI Cards and Charts.
- Created an interactive dashboard for business decision-making.
